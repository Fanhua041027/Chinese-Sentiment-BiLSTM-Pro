# 实验运行流程说明

本目录存放项目各阶段的运行日志，用于记录实验的完整执行过程和输出结果。

## 日志文件说明

| 文件名             | 对应子命令                | 说明                                     |
|--------------------|---------------------------|------------------------------------------|
| `ini_data.txt`     | `python main.py init_data`     | 数据初始化日志（清洗、划分 train/val/test） |
| `proprecess.txt`   | `python main.py preprocess`    | 词表构建日志                             |
| `train.txt`        | `python main.py train`         | Bi-LSTM + Attention 训练日志（20 个 epoch） |
| `predict.txt`      | `python main.py predict`       | Bi-LSTM 测试集评估与 Case Study 可视化日志 |

| `baseline_bert.txt`| `python main.py baseline_bert` | BERT 微调训练日志（Docker 容器内运行）   |
| `plot_curve.txt`   | `python main.py plot_curve`    | 训练曲线绘制日志                         |
| `plot_bert_cm.txt` | `python main.py plot_bert_cm`  | BERT 混淆矩阵绘制日志（Docker 容器内运行）|

## 完整运行顺序

建议按以下顺序从零开始运行整个实验流程：

### 步骤 1：数据预处理（init_data）

```bash
poetry run python main.py init_data
```

**预期输出**：
- `dataset/processed/train.csv`（训练集，约 104 万条）
- `dataset/processed/val.csv`（验证集）
- `dataset/processed/test.csv`（测试集，约 29.7 万条）
- `dataset/processed/meta.json`（数据集统计信息）
- `dataset/figures/length_distribution.png`（文本长度分布图）

### 步骤 2：构建词表（preprocess）

```bash
poetry run python main.py preprocess
```

**预期输出**：
- `dataset/processed/vocab.pkl`（词表文件，包含 30,000 个 token）

### 步骤 3：训练 Bi-LSTM（train）

```bash
poetry run python main.py train
```

**预期输出**：
- `checkpoints/best_model.pth`（验证集 F1 最优的模型参数）
- `dataset/reports/training_log.csv`（训练日志）

### 步骤 4：评估 Bi-LSTM + 可视化（predict）

```bash
poetry run python main.py predict
```

**预期输出**：
- `dataset/reports/test_report.txt`（测试集分类报告）
- `dataset/figures/confusion_matrix.png`（混淆矩阵）
- `dataset/figures/case_study_1.png` 等（注意力热力图）

### 步骤 5：绘制训练曲线（plot_curve）

```bash
poetry run python main.py plot_curve
```

**预期输出**：
- `dataset/figures/training_curve.png`

### 步骤 6：Naive Bayes 基线（baseline_nb）

```bash
poetry run python main.py baseline_nb
```

**预期输出**：
- `dataset/reports/baseline_report.txt`
- `dataset/figures/baseline_confusion_matrix.png`

### 步骤 7：BERT 微调（baseline_bert）- Docker 容器内运行

```bash
docker build -t sentiment-bert .
docker run --gpus all --ipc=host -it --rm -v "$(pwd)":/app sentiment-bert \
    python main.py baseline_bert
```

**预期输出**：
- `checkpoints/bert_finetuned.pth`（BERT 微调后模型）
- `dataset/reports/bert_report.txt`（BERT 分类报告）
- `dataset/figures/model_comparison.png`（Bi-LSTM vs BERT 对比图）

### 步骤 8：BERT 混淆矩阵（plot_bert_cm）- Docker 容器内运行

```bash
docker run --gpus all --ipc=host -it --rm -v "$(pwd)":/app sentiment-bert \
    python main.py plot_bert_cm
```

**预期输出**：
- `dataset/figures/bert_confusion_matrix.png`

## 注意事项

1. **本地 vs Docker**：
   - 步骤 1-6 在本地运行（使用 Poetry 管理的虚拟环境）
   - 步骤 7-8 在 Docker 容器内运行（解决 CUDA 兼容性问题）

2. **日志文件为只读**：
   - 本目录下的 `*.txt` 日志文件是实际运行的原始记录
   - 请勿修改或删除，仅供参考和复现验证

3. **GPU 要求**：
   - Bi-LSTM 训练建议使用 GPU（约 50 分钟）
   - BERT 微调必须使用 GPU（约 12 小时）
