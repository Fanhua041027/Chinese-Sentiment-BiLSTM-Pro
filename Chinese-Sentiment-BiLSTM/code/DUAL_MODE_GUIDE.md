# 双模式情感分析系统使用指南

## 🎯 功能概述

"情感分析"页面现在提供**双模式**情感分析功能：
- **🚀 本地模型（Bi-LSTM + Attention）**：快速、隐私保护、无需联网
- **🤖 DeepSeek AI**：功能丰富（关键词、摘要）、准确率高

## 📍 页面位置

所有情感分析功能都**仅在"情感分析"标签页**显示，包括：
- ✅ 分析模式选择（本地模型/DeepSeek AI）
- ✅ 单条文本情感分析
- ✅ 📊 批量文件分析（CSV/Excel）

**其他页面（如"模型性能"、"可视化分析"等）不会显示这些功能。**

## 🚀 使用步骤

### 方法一：单条文本分析

1. **打开网页**：
   - 访问：http://localhost:8000/Chinese-Sentiment-BiLSTM/code/advanced_app.html
   - 点击导航栏的"**情感分析**"标签

2. **选择分析模式**：
   - 点击"**🚀 本地模型（Bi-LSTM）**"或"**🤖 DeepSeek AI**"

3. **输入文本**：
   - 在文本框中输入评论（至少 5 个字符）
   - 例如："这是我今年看过的最好的电影，特效炸裂，强烈推荐！"

4. **开始分析**：
   - 点击分析按钮
   - 等待分析完成
   - 查看结果

### 方法二：批量文件分析

1. **准备文件**：
   - CSV 格式或 Excel 格式
   - 文件应包含 `text` 列
   - 每行是一条待分析的文本

2. **上传文件**：
   - 在"📊 批量文件分析"区域
   - 点击"上传 CSV 或 Excel 文件"
   - 选择您的文件

3. **开始分析**：
   - 点击"开始批量分析"
   - 等待分析完成
   - 查看结果表格

## 📊 测试文件示例

### CSV 文件格式

```csv
text
这是我今年看过的最好的电影，特效炸裂，强烈推荐！
```

### Excel 文件格式

创建 Excel 文件，第一列为 `text`，每行是一条待分析的文本。

## 🔧 服务启动

### 启动本地模型服务

```bash
cd Chinese-Sentiment-BiLSTM\code
python local_model_api.py
```

**注意**：需要确保已训练模型并生成以下文件：
- `dataset/processed/vocab.pkl` - 词表文件
- `checkpoints/best_model.pth` - 模型权重

### 启动 DeepSeek AI 服务

```bash
cd Chinese-Sentiment-BiLSTM\code
python deepseek_api.py
```

**注意**：需要设置环境变量 `DEEPSEEK_API_KEY`

### 启动网页服务器

```bash
python -m http.server 8000
```

## 🎯 功能对比

| 特性 | 本地模型 | DeepSeek AI |
|------|---------|-------------|

## ⚠️ 常见问题

### Q1: 批量分析功能在其他页面显示了怎么办？

**解决方案**：
- 已修复 HTML 结构，批量分析功能现在只在"情感分析"标签页显示
- 如果仍有问题，请清除浏览器缓存（Ctrl+Shift+Delete）

### Q2: 本地模型服务无法启动？

**检查步骤**：
1. 确认模型文件存在：
   ```bash
   dir dataset\processed\vocab.pkl
   dir checkpoints\best_model.pth
   ```
2. 如果文件不存在，需要先训练模型：
   ```bash
   python src/train.py
   ```

### Q3: 点击分析按钮没有反应？

**解决方案**：
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误
3. 确认对应的后端服务已启动
4. 检查网络连接（如果使用 DeepSeek AI）

## 📝 输出说明

### 本地模型输出

- **情感倾向**：正面/负面
- **置信度**：0-100%
- **分析时间**：秒数
- **情感得分**：进度条显示

### DeepSeek AI 输出

- **情感倾向**：正面/负面/中性
- **情感分数**：0-100
- **关键词**：3 个关键词
- **智能摘要**：一句话总结
- **分析时间**：秒数
